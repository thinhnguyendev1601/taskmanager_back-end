# taskmanager_back-end

1. use the data.sql to create your own local database on mySQL Workbench
2. then change the password to your own local SQL server in the application.properties(.yml)
3. check for Java version in the pom.yml (Java 25 for this project), make sure to have maven downloaded: https://maven.apache.org/download.cgi
4. in your terminal: type 'mvn clean install' then 'mvn spring-boot:run' or if they're just some changes you have in the code, you can use 'mvn clean compile' instead of 'mvn clean install'
5. it is on localhost port 8080
